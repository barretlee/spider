Spider
===

A Web Spider. Grabbing article, images, audio and video in multiple processes.

### Usage

- rename `_config.js` to `config.js`

- set config file
```js
var OPEN_ARTICLE = true;
var OPEN_IMG = true;
var OPEN_VIDEO = true;
```

- start
```bash
node index.js
```
